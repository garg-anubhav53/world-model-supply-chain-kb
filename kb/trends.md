## T1 — Humanoid actuator ramp (prototype → pilot production, 2026)
Buyers: Figure (BotQ ~1 robot/90min), Apptronik, Neura, Unitree. Forces: actuators, strain-wave reducers, frameless motors + laminations, force-torque sensors, joint encoders, NdFeB magnets.
Un-crowded links: gear machine-tools (make the reducers), encoder ICs (Melexis), motor laminations (Mitsui), ex-China magnets (Neo).

## T2 — Ex-China magnet mandate (rare-earth export controls + EU CRMA)
Forces Western robot/EV/AV motor makers to source non-China NdFeB + heavy-RE (Dy/Tb). Implicated: Neo (NEO.TO), recyclers (MKA/IXR/UCU). Risk: rare-earth price swings; China de-escalation.

## T3 — Robot-joint position sensing (every DoF needs an encoder)
Content scales with joint count (humanoid = 28–44 actuators). Implicated: Melexis inductive encoders; competes with iC-Haus/RLS/Allegro. Test NEO-specific / MLX-specific capture, not just "encoders needed."

## T4 — Hyundai Motor Group / Boston Dynamics robotics-and-AV ecosystem (Korea)
Hyundai owns Boston Dynamics and is building its own mobile-robot platform "MobED" plus humanoid ambitions. Forces Korean tier-2/3 suppliers via disclosed co-development deals. Implicated: SOS Lab (464080, LiDAR, WATCH — top idea), SBB Tech (389500, reducer/drive module, already re-rated +101%, killed), SPG (058610, reducer, already large/known). Un-crowded angle: SOS Lab is the freshest, smallest, least-priced name in this cluster — watch for MobED 1H2026 mass-production revenue actually landing.

Implicated: HSAI (WATCH, already ran+pulled back on US 1260H litigation risk, see kb/companies/HSAI.md).

## T9 — In-house vs. merchant component sourcing as the master predictor (Fire 4 synthesis)
Sharpens T1/T4/T5: across every buyer-BOM teardown run so far (Figure, Apptronik, Agility, Waymo = dead ends; Zoox, Hyundai MobED, Chinese humanoid OEMs, Tesla Optimus-candidate = productive), the split is NOT US-vs-China or captive-startup-vs-not — it's whether that SPECIFIC buyer builds the component in-house. Use this as the first filter before starting any new buyer-BOM teardown: check whether the buyer has publicly stated vertical integration for the target component before spending search budget on it.
